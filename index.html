Javascript

The frist thing you want to do is see if your computer has javascript. If it does, then you can go to the next step. The second thing you want to do is find a code translator, so you can see the result. If you have all of these, then you can start programming.

Basic Shapes

Ok, lets get started. Open up your code editor, and we will start.

The first thing we will learn is how to draw a circle. To draw a circle, type the command ellipse, and make sure it is lowercase.

Didn't work, did it? well that is because we didn't type in enough information. Now try typing ellipse(200,200,20,20);if you have a 400x400 canvas, then a circle should appear in the middle of the screen. If it still didn't work, check your spelling and make sure it is lowercase.
